enum class m0st4fa::fsm::FSM_MODE#

The simulation mode that simulation functions use.

See also

NonDeterFiniteAutomaton<TransFuncT, InputT>::simulate(const InputT& input, FSM_MODE mode) const;

See also

DeterFiniteAutomaton<TransFuncT, InputT>::simulate(const InputT& input, FSM_MODE mode) const;

Values:

enumerator MM_WHOLE_STRING#

The simulation returns true if and only if the whole string accepts.

enumerator MM_LONGEST_PREFIX#

Look for the longest prefix only. Once found, the string accepts.

enumerator MM_LONGEST_SUBSTRING#

Look for the longest substring, which might be the entire string.

enumerator MM_NONE#

The default value.

enumerator MM_FSM_MODE_COUNT#

The number of enumerators that this enumeration has.

enumerator MT_EPSILON_NFA#

This sets the NFA that the NonDeterFiniteAutomaton object simulates to an epsilon NFA.

enumerator MT_NON_EPSILON_NFA#

This sets the NFA that the NonDeterFiniteAutomaton object simulates to a non-epsilon NFA.

enumerator MT_DFA#

The type of a DeterFiniteAutomaton. This is set automatically for every DeterFiniteAutomaton object created.

The number of enumerators that this enumeration has.
